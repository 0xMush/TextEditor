# ✨ HolyTextEditor

[![Language](https://img.shields.io/badge/Language-C-blue.svg)](https://en.wikipedia.org/wiki/C_(programming_language))

**HolyTextEditor** is a lightweight, terminal-based text editor written in C. Simple, fast, and focused — just a text editor, nothing more, nothing less.

## 📝 About

This project started as a minimalistic text editor with a straightforward goal: provide a distraction-free writing or coding environment directly in your terminal. It's built for simplicity and speed.

## ✨ Features (Planned / Implemented)

- Basic text editing and navigation
- Terminal-based interface (using raw mode)
- Written entirely in C for performance
- Minimal dependencies

> **Note:** This is an actively developed project. Some features may be experimental or incomplete.

## 🚀 Getting Started

### Prerequisites

- A C compiler (gcc, clang, etc.)
- Make (optional, but recommended)

### Building from Source

Clone the repository and build the editor:

```bash
git clone https://github.com/0xMush/HolyTextEditor.git
cd HolyTextEditor
make
```
